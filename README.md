# vue-todo

## Create Vue application with Vue CLI

```bash
npx create-vue myApp
```

Simple todo app using Vue.js

## Type-check note

- `src/features/error-page/views/ErrorPage.vue` intentionally contains type errors for demo/testing purposes.
- That file uses `// @ts-nocheck`, so `npm run type-check` ignores its diagnostics by design.

# PrimeVue installation

```bash
npm install primevue @primevue/themes
```

# TailwindCSS installation

```bash
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
```
